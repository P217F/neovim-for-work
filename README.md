# neovim-for-work
This is an dotfiles repo for neovim  

## Installation
- Cloning to ~/.config/nvim: git clone https://github.com/P217F/neovim-for-work ~/.config/nvim  
- Open neovim and using it  

### Requirements:
- clangd
- OpenDebugAD7(providing by cpptools)
- pyright
- debugpy(require virtual environment python)
- flake8(require virtual environment python)
- pytest(require virtual environment python)
- typescript-language-server
- eslint
- npm
